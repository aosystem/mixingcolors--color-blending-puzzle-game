# Mixing Colors: Color Blending Challenge 🎨🧪

A sophisticated and engaging **Color-Mixing Puzzle** built with **Flutter**. This application challenges players to recreate specific target shades by blending primary and secondary colors, testing their artistic intuition and color perception.

## 🌟 Key Features
- **Dynamic Color Blending:** Realistic color mixing logic that simulates how pigments or light interact.
- **Progressive Difficulty:** Starts with simple primary colors and evolves into complex, multi-layered shades.
- **Color Sense Training:** Perfect for artists, designers, and students to sharpen their eye for detail.
- **Global Support:** Fully localized in **50+ languages**, making the world of color accessible to everyone.

## 💡 Best For...
- **Artists & Designers:** Practicing color theory and observation in a gamified way.
- **Creative Minds:** Anyone who enjoys the satisfying process of mixing and creating new hues.
- **Educational Use:** Teaching the basics of color composition and the relationship between different shades.

## 🛠 Technical Overview
- **Framework:** Flutter
- **Language:** Dart
- **Logic:** Custom color math algorithms for RGB/HSV interpolation and "distance" calculation to determine how close a mix is to the target.

*Note: This repository contains the core logic and source code. Assets (images, sound effects, and level data) are excluded to protect intellectual property.*
